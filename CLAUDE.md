# Simple Circle Showdown (S.C.S. AI) — コード

1v1 観戦オートバトラーの**テキストベース・ターン制プロトタイプ（フェーズ1）**。
プレーヤーは戦闘中操作せず、事前に「人格（10大パラ）」を設計。人格→24小パラ/HP/武器に派生し、フル2Dシムで自動戦闘した結果を **距離＋HP＋テキストログ** で観戦する。

## 構成
- `js/rng.js` — シード付き決定論乱数（同人格＋同seed→同戦闘）
- `js/data.js` — 設計書の確定数値（10大パラ・24小パラ重み行列・武器・HP式・CPUプリセット・**戦場ARENAS(5)・地形TERRAIN**・sim定数）
- `js/derive.js` — 人格 → 24小パラ / HP / 武器選択 ＋ 認知パラメータ(cog)（非線形F・静的相互作用）
- `js/sim.js` — **フル2D空間シム核 ＋ 思考エンジンv2（Stage1+2）**：移動×攻撃チャンネル／多因子評価関数（重み=24小パラ、勝てる間合いwinDist・射撃好機engage 等）／先読み探索（最大深度4）＋モンテカルロ（別系列think-rng）／相手モデリング・学習（手の頻度予測・D6痛打距離回避、順応人格は先読みの相手手に反映）／プランのコミット（規律でヒステリシス）／**戦場ランダム選択＋地形効果（命中/被ダメ/移動/winDistに作用）**／描写v2[戦場/状況(形勢/有利間合い/地形)/思考(プラン/深度/MC/次善/相手の癖)/行動]。決定論（探索は期待値で乱数非消費、MCは別rng、戦場はseed由来）。描画に非依存
- `js/ui.js` — テキスト描画＋人格設定UI＋[NEXT STEP]/[VIEW PARAMETERS]
- `index.html` / `style.css` — ターミナル(CRT)調シェル

## アーキテクチャ方針
**シム核を描画から分離**（テキスト版＝シム核＋テキスト描画／将来のリアルタイム版＝同シム核＋Canvas）。
シムは**フル2D**のまま。表示が距離＋HP＋ログに削がれるだけ。24小パラの再解釈はしない。

## sim定数（実装レベル・チューニング対象）
`data.js` の `SIM`：アリーナ100×40、障害物2個、初期配置、ターン上限30、基本移動量、連射はfireRateを確率反映 等。
設計書で「実装時に詰める／実戦チューニング前提」とされた値。変えたら設計書「実装メモ」へ反映。
