# S.C.S. AI — Simple Circle Showdown（テキストプロトタイプ）

操作しない 1v1 オートバトラー。**戦闘中は一切操作できない**——代わりに事前に「人格」を設計し、その人格から導かれた AI が自動で戦う様子を**観戦・分析**する実験ゲーム。これはその、テキストベース・ターン制プロトタイプ。

## 遊び方
1. 画面下の **AI CONFIG** で、あなたの人格を **10軸 × 各4択**で設定（闘争心・リスク選好・冷静さ・忍耐・規律・順応性・誇り・非情さ・自信・好奇心）。
2. **戦場**（ランダム可）と **CPU**（8アーキタイプ or ランダム人格）を選び、**新規対戦**。
3. **NEXT STEP** で1ターンずつ、または **自動実行** で連続観戦。
4. 決着すると **戦闘分析** が自動表示。命中率・与/被ダメ・間合いの取り方・状態異常などの「戦いぶり」と、**次にどこを直すかの方向性**が出る。
5. 人格を調整して再戦 → 「設計 → 観戦 → 分析 → 再設計」を回すのが核。

## 特徴
- **人格(10大パラ) → 24の挙動パラメータ**を非線形変換で導出（＝トレードオフ設計が"妙"）。**内部の数値は非公開**。戦いぶり（結果）から推し量る。
- **武器も人格依存**：遠距離9種＋近距離8種から自動選択。リロード/会心/チャージ/**状態異常6種**（燃焼・出血・毒・麻痺・脆弱・鈍足）など。
- **戦場＆地形**：5つの戦場をランダム選択。茂み・瓦礫・沼地・高所が命中/回避/移動に影響＝同じ人格でも戦場で勝敗が変わる。
- **思考エンジン**：多因子評価関数＋先読み＋モンテカルロ＋相手モデリング。人格が「思考の深さ」まで決める。
- **一期一会**：既定は毎回ランダム（非決定論）。🎲OFF＋seed固定で同じ戦闘を再現も可能。

ビルド不要の静的サイト。`index.html` を開くだけで動く。設計の正本は Obsidian の「ゲーム設計書.md」（詳細は `CLAUDE.md`）。
